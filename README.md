# ES-1-outcome
ES-1-outcome (Endogenous Separation conditioned on the previous outcome) is a repository with various tools to formally analyze the mechanism of endogenous separation (or conditional dissociation) in the evolutionary emergence of cooperation.

The repository contains two main folders:
  * **netlogo-model** contains an agent-based NetLogo model, which can be run with [NetLogo Desktop](https://ccl.northwestern.edu/netlogo/). You can find its full description at [https://luis-r-izquierdo.github.io/ES-1-outcome/](https://luis-r-izquierdo.github.io/ES-1-outcome/)
  * **mathematica-supporting-files** contains four Mathematica notebooks that are very useful to understand the mechanism of endogenous separation in the evolutionary emergence of cooperation.
